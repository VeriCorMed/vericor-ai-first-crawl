# Mobile Incident Command Center

**Summary:** A self-contained case that establishes command and control quickly and easily. This mobile station comes with features found in typical command centers, with room to organize/ready supplies for the next logistical, operational, or command situation.

The Mobile Incident Command Station (MC-MICC) enables hospitals to quickly and easily establish command and control from one centralized mobile command station. Organize and store all incident supplies within this compact, mobile unit. It features an adjustable laptop stand, overhead light, data jacks, inlay map/doc grease board, two Easyboards®, a wide wheel base, detachable fold-out table, and is fully lockable. Be prepared the next time an incident happens and a mobile solution is needed.

### Technology & Features

-

Integrated overhead light for a bright work space.

-

Adjustable laptop stand for computers or tablets.

-

Two Easyboards® for whiteboard/document visibility.

### Additional Features

* Integrated work surface data and power hub.
* Shelving has built-in electrical.
* Stainless steel drawers.
* HICS organizational chart included.
* Front cover converts into a table.

### Dimensions & Specs

**Closed:** 50″ x 43″ x 25″
**Weight:** 269 lbs.
