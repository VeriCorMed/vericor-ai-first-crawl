# Hospital Incident Command Station

**Summary:** A mobile station used for incident command. This compact unit rolls anywhere, and then by undoing a couple of latches the case opens up into an incident command workstation. Complete with electrical wiring and its own table, this unit is ready.

The Hospital Incident Command Station (MC-HICS) is a mobile case that enables hospitals to quickly and easily establish command and control from one centralized mobile command station. Organize and store all incident supplies within this compact, mobile unit. It features an adjustable laptop stand, overhead light, data jacks, inlay map/doc grease board, two Easyboards®, a wide wheel base, detachable fold-out table, and is fully lockable. Be prepared the next time an incident happens and a mobile solution is needed.

### Technology & Features

- Integrated overhead light for a bright work space.![](https://www.vericormed.com/wp-content/uploads/2015/10/green-clipart-circle-2-150x150.png)Adjustable laptop stand for computers or tablets.![](https://www.vericormed.com/wp-content/uploads/2015/10/green-clipart-circle-3-150x150.png)Two Easyboards® for whiteboard/document visibility.

### Additional Features

* Integrated work surface data and power hub.
* Inlay map/document grease board.
* Shelving has built-in electrical.
* Stainless steel drawers.
* HICS organizational chart included.
* Front cover converts into a table.

### Dimensions & Specs

**Closed:** 50″ x 43″ x 25″
**Weight:** 295 lbs.
