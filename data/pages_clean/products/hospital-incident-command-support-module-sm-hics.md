# Support Module for MC-HICS

**Summary:** A standard, all-hazard support module that readies a Hospital Incident Command Station for both emergency and non-emergency situations. These items are necessary for an efficient response in the wake of an incident.

The Support Module for MC-HICS (MC-HICSSM) is a kit used to efficiently establish command and control. These necessary supplies support a Hospital Incident Command Station in the wake of an incident. They include HICS forms, HICS pocket guide, HICS job action sheets, incident command vests, clipboards, pens, markers, tape and more.

### Dimensions & Specs

**Boxed:** 20” x 16” x 4”
**Weight:** 10 lbs.
