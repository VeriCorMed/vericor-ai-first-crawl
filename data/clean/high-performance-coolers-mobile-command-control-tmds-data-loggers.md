#### Simple & Effective
**Temp System:** PCM (phase change material)  
* Optimum temperature stabilization without needing ice, electricity or buffering material.  
  
**Insulation:** VIP (vacuum insulated panels)  
* Maximum size-to-space ratio to decrease both bulk and weight.   
  
**Case:** fabric or hard-shell (depends on model)  
* Unique features on the outside of the case enhance the user experience.

#### Hospital Incident Command Station
In preparation for the next logistical, operational or command situation, organize needed supplies within a colorful Hospital Incident Command Station by VeriCor. In storage it takes up the space of a desk. In deployment it facilitates command and control. A built-in laptop stand, overhead light, inlay map/document greaseboard, and two Easyboards® are just a few of the features of this case. Pair it with the [HICS Support Module](https://www.vericormed.com/product/hospital-incident-command-support-module-sm-hics/) (or your own kit) and be ready to respond anywhere…anytime!

### TMDs (Data Loggers)[![Simple and Easy to Use Data Loggers](https://www.vericormed.com/wp-content/uploads/2018/01/Simple-and-Easy-to-Use-Data-Loggers-300x200.jpg)](https://www.vericormed.com/wp-content/uploads/2018/01/Simple-and-Easy-to-Use-Data-Loggers.jpg)
#### Simple and Easy to Use
As the guidelines for monitoring temperature sensitive product become more stringent, an emphasis is being placed on data loggers and the ability to create a detailed report of the storage environment. VeriCor carries a variety of loggers that have the following characteristics:  
* individually calibrated  
* programmable logging interval  
* buffered temperature probe(s)  
* alarm  
* data as a simple spreadsheet (no software needed)  
* easy to buy (direct or via Amazon)  
Need help choosing? Use the “which model do I need” pdf [here](https://d2zhgehghqjuwb.cloudfront.net/accounts/1351/original/Temperature_Monitor_Overview.pdf?1519836596).
