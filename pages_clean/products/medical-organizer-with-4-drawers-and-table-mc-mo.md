# Medical Organizer w/ 4 Drawers & Table

**Summary:** A mobile case that is a storage container, cabinet, table and medical case all-in-one. This exclusive, durable case has partitioned drawers for organizing supplies and label holders for identifying what is in each drawer.

[vc\_row][vc\_column][vc\_column\_text]The Medical Organizer w/ 4 Drawers & Table (MC-MO) is a storage container, cabinet, table and medical case all-in-one. This durable case has partitioned drawers for easy, in-field access to vital medical gear. The detachable table provides a much needed work surface. It features deep drawers, customizable drawer dividers, easy-slide drawer guides, drawer label holders, butterfly latches, and a wide wheel base.[/vc\_column\_text][/vc\_column][/vc\_row][vc\_row full\_width="stretch\_row\_content\_no\_spaces"][vc\_column][vc\_separator border\_width="2"][/vc\_column][/vc\_row][vc\_section][vc\_row][vc\_column][vc\_column\_text]
