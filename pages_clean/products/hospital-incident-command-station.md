# Hospital Incident Command Station

**Summary:** A mobile station used for incident command. This compact unit rolls anywhere, and then by undoing a couple of latches the case opens up into an incident command workstation. Complete with electrical wiring and its own table, this unit is ready.

[vc\_row][vc\_column][vc\_column\_text]The Hospital Incident Command Station (MC-HICS) is a mobile case that enables hospitals to quickly and easily establish command and control from one centralized mobile command station. Organize and store all incident supplies within this compact, mobile unit. It features an adjustable laptop stand, overhead light, data jacks, inlay map/doc grease board, two Easyboards®, a wide wheel base, detachable fold-out table, and is fully lockable. Be prepared the next time an incident happens and a mobile solution is needed.[/vc\_column\_text][/vc\_column][/vc\_row][vc\_row full\_width="stretch\_row\_content\_no\_spaces"][vc\_column][vc\_separator border\_width="2"][/vc\_column][/vc\_row][vc\_section][vc\_row][vc\_column][vc\_column\_text]
