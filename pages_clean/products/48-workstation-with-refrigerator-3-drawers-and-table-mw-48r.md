# 48” Workstation w/ 3 Drawers, Table & Refrigerator

**Summary:** A mobile workstation with three drawers for storage, a refrigerator for vaccine (or other temperature sensitive product), and a detachable table for working on. Undo a couple latches and it opens up, ready for work.

[vc\_row][vc\_column][vc\_column\_text]The 48” Workstation w/ 3 Drawers, Table & Refrigerator (MC-48R) is a mobile workstation for medical supply storage and mobility. This self-contained case on wheels has ample storage, a built-in refrigerator, transport handles and a detachable table that can move when and where it is needed. It is lockable for secure storage, has an adjustable refrigerator without freezer, glove holders in the lid for easy access, and easy lock wheels with a wide wheel base for easy/safe maneuverability.[/vc\_column\_text][/vc\_column][/vc\_row][vc\_row full\_width="stretch\_row\_content\_no\_spaces"][vc\_column][vc\_separator border\_width="2"][/vc\_column][/vc\_row][vc\_section][vc\_row][vc\_column][vc\_column\_text]
