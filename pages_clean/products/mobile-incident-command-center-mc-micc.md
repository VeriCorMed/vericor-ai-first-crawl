# Mobile Incident Command Center

**Summary:** A self-contained case that establishes command and control quickly and easily. This mobile station comes with features found in typical command centers, with room to organize/ready supplies for the next logistical, operational, or command situation.

[vc\_row][vc\_column][vc\_column\_text]The Mobile Incident Command Station (MC-MICC) enables hospitals to quickly and easily establish command and control from one centralized mobile command station. Organize and store all incident supplies within this compact, mobile unit. It features an adjustable laptop stand, overhead light, data jacks, inlay map/doc grease board, two Easyboards®, a wide wheel base, detachable fold-out table, and is fully lockable. Be prepared the next time an incident happens and a mobile solution is needed.[/vc\_column\_text][/vc\_column][/vc\_row][vc\_row full\_width="stretch\_row\_content\_no\_spaces"][vc\_column][vc\_separator border\_width="2"][/vc\_column][/vc\_row][vc\_section][vc\_row][vc\_column][vc\_column\_text]
