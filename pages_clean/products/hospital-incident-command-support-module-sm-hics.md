# Support Module for MC-HICS

**Summary:** A standard, all-hazard support module that readies a Hospital Incident Command Station for both emergency and non-emergency situations. These items are necessary for an efficient response in the wake of an incident.

[vc\_row][vc\_column][vc\_column\_text]The Support Module for MC-HICS (MC-HICSSM) is a kit used to efficiently establish command and control. These necessary supplies support a Hospital Incident Command Station in the wake of an incident. They include HICS forms, HICS pocket guide, HICS job action sheets, incident command vests, clipboards, pens, markers, tape and more.[/vc\_column\_text][/vc\_column][/vc\_row][vc\_row full\_width="stretch\_row" content\_placement="middle" css=".vc\_custom\_1538504489259{background-image: url(https://www.vericormed.com/wp-content/uploads/2015/10/Dimensions-Background.png?id=6494) !important;background-position: center !important;background-repeat: no-repeat !important;background-size: cover !important;}"][vc\_column][vc\_column\_text]
