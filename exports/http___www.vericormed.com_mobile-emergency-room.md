# Mobile ER System
![Mobile ER header](https://www.vericormed.com/wp-content/uploads/2022/08/Mobile-ER-header.png)
An unforeseen incident can overwhelm a healthcare system, and it may even require an expansion of the delivery system to one that includes the role of a temporary emergency room. A VeriCor Mobile Emergency Room (ER) System is a complete treatment system that provides advanced levels of care during mass casualty incidents and trauma. VeriCor’s ER System is mobile and robust enough that it can be established near an emergency, or it can expand an existing medical facility in the event of a sudden patient surge. This modular-based system allows customers to build a Mobile ER to their own specifications/requirements. The system is swift to mobilize and deploy, intuitively organized so staff can access vital equipment and supplies efficiently, and quick to demobilize at the end of the training exercise or operation.
Take a look at the information below, but if you are remotely interested in this system be sure to call (608) 526-6901 and ask for a Mobile ER System Specialist. Or send an email to info@vericormed.com with your contact information and they will be sure to get back to you.

**WHO** IS A MOBILE ER SYSTEM FOR?  
Any health care facility or coalition needing an emergency preparedness system for treating a surge of patients.
**WHAT** IS A MOBILE ER SYSTEM?  
A system of “modules” containing supplies needed to treat a surge of patients.
28 standard modules are available à la carte. Pick the functions desired to create a unique system.
Each module contains supplies specific to that type of care.
Supplies are organized within labeled medical cases so that emergency care providers will know exactly where an item is and where they can go to get more.
_The SmartBook™ brings it all together._  
A printed resource containing the system details including pictures and line item lists of quantities, locations, lot #s, expiration dates, etc. A CD containing digital files also included.
**WHEN** IS A MOBILE ER SYSTEM USED?  
During a MCI patient surge when a temporary emergency room is needed to provide casualties advanced levels of care.
**WHERE** CAN A MOBILE ER SYSTEM BE USED?  
Any building of opportunity or temporary shelter (tents).
**WHY** BUY A MOBILE ER SYSTEM?  
It’s mobile.  
It’s complete.  
It’s efficient.  
It’s simple.
